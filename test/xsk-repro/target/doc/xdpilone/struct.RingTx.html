pub fn as_raw_fd(&self) -> c_int

Get the raw file descriptor of this TX ring.

§Safety

Use the file descriptor to attach the ring to an XSK map, for instance, but do not close it and avoid modifying it (unless you know what you’re doing). It should be treated as a BorrowedFd<'_>. That said, it’s not instant UB but probably delayed UB when the RingTx modifies a reused file descriptor that it assumes to own (for instance, wake sends a message to it).
