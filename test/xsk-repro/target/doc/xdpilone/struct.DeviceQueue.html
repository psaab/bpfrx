Struct DeviceQueue

pub struct DeviceQueue { /* private fields */ }
Expand description

One device queue associated with an XDP socket.

A socket is more specifically a set of receive and transmit queues for packets (mapping to some underlying hardware mapping those bytes with a network). The fill and completion queue can, in theory, be shared with other sockets of the same Umem.

pub fn as_raw_fd(&self) -> c_int

Get the raw file descriptor of this ring.

§Safety

Use the file descriptor to attach the ring to an XSK map, for instance, but do not close it and avoid modifying it (unless you know what you’re doing). It should be treated as a BorrowedFd<'_>. That said, it’s not instant UB but probably delayed UB when the DeviceQueue modifies a reused file descriptor that it assumes to own.
