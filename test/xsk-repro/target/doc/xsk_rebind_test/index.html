Minimal AF_XDP zero-copy rebind test for xdpilone on mlx5.

Tests whether XSK zero-copy receive works after a link DOWN/UP cycle.

Prerequisites:

  • An XDP program must be attached to the interface that redirects to an xskmap
  • The xskmap fd must be provided via –xsk-map-fd
  • Traffic must be flowing to the interface (e.g. ping from another host)

Usage: xsk-rebind-test –xsk-map-fd
